# AnimeEncount
アニメエンカウント

## 機能
2014年以降から現在まで放映されたアニメをどれか1つランダムで表示する

## 使用言語
Go

## 使用API
アニメ作品の情報を返すREST形式のAPIサーバー

[ShangriLa Anime API](https://github.com/Project-ShangriLa/sora-playframework-scala)
